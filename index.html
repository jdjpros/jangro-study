    📊 진행률 / 복습 뱃지
    진행률 바: 전체 문제 중 한 번이라도 풀어본 문제 비율입니다.

    오늘 복습 N문제: 오늘 복습 예정일이 된 문제 수입니다. 0이 되면 오늘 학습 완료!
    🔄 재풀기 기능
    각 과목 카드 하단의 [🔄 재풀기] 버튼을 누르면 해당 과목의 모든 문제가 오늘 다시 출제됩니다.

    ✅ 정답/오답 이력은 그대로 유지됩니다.
    이전에 틀렸던 문제를 다시 맞히면 알았음 횟수가 쌓이고, 복습 주기도 자동으로 늘어납니다.

    재풀기 후 [🔔 복습 필요 문제 모아보기]를 누르면 바로 시작할 수 있습니다.
    📅 회독 설정
    헤더의 [📅 회독 설정] 버튼을 누르면 목표 기간을 설정할 수 있습니다.

    예) 30을 입력하면 → 오늘부터 30일 안에 전 과목 1회독을 목표로 하루 목표 문제 수를 자동 계산합니다.

    각 과목 카드에 📌 오늘 목표 N문제가 표시되어 하루 학습량을 쉽게 확인할 수 있습니다.

    회독 설정 메뉴에서 과목별 초기화(🗑️)도 함께 관리할 수 있습니다.
    📱 홈 화면에 추가하는 방법
    삼성 인터넷:
    하단 메뉴(☰) → 페이지 추가 → 홈 화면에 추가

    Chrome:
    우측 상단 메뉴(⋮) → 홈 화면에 추가
    📅 회독 설정
    오늘부터 며칠 안에 1회독을 완료할지 입력하세요.
    기간을 입력하면 과목별 하루 목표 문제 수가 표시됩니다.
